# 📷 Telegram Image to PDF Bot

A Telegram bot that converts multiple images into a single PDF file.

---

## 🚀 Features

* 📤 Upload multiple images
* 📄 Convert images into a single PDF
* ⚡ Fast and simple processing
* ☁️ Deployed on cloud for real-time usage

---

## 🛠️ Tech Stack

* Python
* python-telegram-bot
* img2pdf

---

## ⚙️ How It Works

1. User sends images to the bot
2. Bot stores images temporarily
3. Images are converted into a PDF
4. PDF is sent back to the user

---

## 📦 Installation (Local Setup)

```bash
git clone https://github.com/YOUR_USERNAME/telegram-pdf-bot.git
cd telegram-pdf-bot
pip install -r requirements.txt
```

---

## ▶️ Run Locally

```bash
export BOT_TOKEN=your_token_here
python bot.py
```

---

## ☁️ Deployment

The bot is deployed using Railway for 24/7 availability.

Steps:

* Connect GitHub repository
* Add environment variable `BOT_TOKEN`
* Deploy the project

---

## 🔐 Environment Variables

| Variable  | Description        |
| --------- | ------------------ |
| BOT_TOKEN | Telegram bot token |
